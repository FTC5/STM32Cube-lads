В проекті Proteus два термінали. Перший для відправлення команд в перший контролер і перегляду виводу другого контролеру, другий термінал для перегляду виводу першого контролеру.
Команди
0 - команда зупинки передачі масиву нескінчених даних.
1 - початок передачі масиву нескінчених даних.
3 - команда зупиняє передачу даних на обох контролерах
4 - команда початку передачі нескінченого масиву даних другим контролером.
Тобто команди 0 і 1 здійснють управління тільки першим контролером. А команди 3 і 4 здійснють управління першим контролером для того щоб він управлял другим контролером.
